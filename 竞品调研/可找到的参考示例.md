# 授权弹窗 — 可找到的真实参考示例

> 美团的功能藏在医美/口腔/婚摄的下单流程深处，确实不好找。以下是真正能直接找到的参考。

---

## 最好找的 4 个参考（按可获取难度排序）

### 1. 微信小程序 · 手机号授权（⭐ 最好找，5分钟可体验）

**在哪看**：打开微信 → 随便进一个需要登录的小程序（如京东、美团、滴滴）→ 点击"微信手机号授权登录"

**你会看到**：
```
┌──────────────────────────────────┐
│                                  │
│  京东购物 申请获取                │
│  你的手机号码                     │  ← 微信原生弹窗
│                                  │
│  用于登录、订单通知等服务          │  ← 用途说明
│                                  │
│  [不允许]          [允许]         │
└──────────────────────────────────┘
```

**获取方式**：
- 打开任意微信小程序（以京东为例）
- 退出登录 → 重新点击登录 → 选择"微信手机号快速登录"
- 或开发阶段：微信开发者工具 → 新建小程序项目 → 使用 `getPhoneNumber` 示例代码直接模拟

**文档**：`https://developers.weixin.qq.com/miniprogram/dev/framework/open-ability/getPhoneNumber.html`

**特点**：微信原生面板，不可自定义UI，但信任度最高。所有小程序都用这一套。

---

### 2. 支付宝 · 手机号授权（⭐ 同样好找）

**在哪看**：打开支付宝 → 进哈啰、淘票票等小程序 → 需要手机号时触发授权

**你会看到**：
```
┌──────────────────────────────────┐
│  哈啰 申请获取你的手机号码         │
│                                  │
│  手机号将用于登录、接收订单通知     │
│                                  │
│  ○ 仅本次允许                     │  ← 有时长选项（微信没有）
│  ○ 始终允许                      │
│                                  │
│  [拒绝]              [同意]       │
└──────────────────────────────────┘
```

**获取方式**：支付宝开发者文档 → `https://opendocs.alipay.com/mini/component/getphonenumber`

**特点**：与微信类似，但多了"仅本次允许/始终允许"的授权时长选择。授权管理页的设计也值得参考（支付宝 → 我的 → 设置 → 隐私 → 授权管理）。

---

### 3. 58同城 · 联系经纪人（⭐⭐ 略微需要操作）

**在哪看**：打开58同城App → 搜一套二手房 → 点进房源详情 → 点底部"联系经纪人"

**你会看到**：
```
┌──────────────────────────────────┐
│  授权[王某某经纪人]获取            │
│  您的手机号                       │
│                                  │
│  该经纪人将电话联系您，             │
│  为您提供房源咨询服务              │
│                                  │
│  [取消]            [确认授权]      │
└──────────────────────────────────┘
```

**获取方式**：直接下载58同城App → 搜北京二手房 → 任意房源 → 点底部"电话咨询"或"在线咨询"

**特点**：一对一授权（具体到某个经纪人），授权后经纪人回拨。与美柚的"授权具体服务商"场景最接近。

---

### 4. 大众点评 · 免费咨询（⭐⭐ 同样需要操作）

**在哪看**：打开大众点评App → 搜"婚纱摄影"或"口腔"→ 进入商家页 → 点"免费咨询"

**你会看到**：
- 表单页（手机号 + 需求描述）
- 底部有《用户授权协议》勾选框
- 提交后商家在后台收到线索

**获取方式**：下载大众点评App → 搜"口腔"→ 进入排名靠前的商家 → 点"免费咨询"或"获取报价"

**特点**：表单式授权（不是纯弹窗），多商家竞价的线索分发模式。可以作为"表单式"授权方案的参考。

## 为什么美团那个不好找？

美团的功能触发条件非常苛刻：

| 条件 | 说明 |
|------|------|
| 必须登录 | 需要美团账号 |
| 必须进入特定品类 | 医美、口腔、婚摄、家装（外卖/到店餐饮没有） |
| 必须在特定页面触发 | 商家详情页的"咨询"或"预约"按钮 |
| 可能被AB实验控制 | 部分用户可能看不到（美团在做灰度） |
| 不同城市可能不同 | 一线城市覆盖率高，下沉市场可能没有 |

**所以不是你找不到，是确实不好找。** 这也是为什么我建议优先参考微信/支付宝的原生组件——它们是公开文档、标准化实现、任何开发者都能直接体验。
