# 调研概述

## 1.1 目的

**背景**：美柚App会员领取福利（如洗牙、体检等）后核销率低，大量福利被领取但未实际使用，造成运营资源浪费和用户体验断裂。

**核心问题**：如何通过"授权弹窗"机制，在用户领取福利后获取其授权，将手机号提供给第三方服务商，由服务商主动联系用户推动核销，从而提升核销率？

**期望达成**：
- 设计一套合规、高效、用户体验良好的会员福利授权弹窗方案
- 明确授权流程、UI交互、状态流转、异常处理、数据埋点等关键设计
- 确保方案符合《个人信息保护法》等法规要求

## 1.2 调研目标

1. 梳理至少8个平台的授权弹窗/手机号共享机制，提取可借鉴的设计模式
2. 通过浏览器实地调研，获取微信/支付宝官方文档、36氪/woshipm行业文章的原始截图和关键观点
3. 对比各平台授权方案的优缺点，识别美柚可借鉴的最佳实践
4. 输出清晰的产品建议，包括授权时机、UI设计、SLA机制、授权管理等维度

## 1.3 调研范围与边界

**涵盖范围**：
- 10个竞品平台的授权机制分析（美团、微信、支付宝、58同城、大众点评、88VIP、京东Plus、平安、招行、携程）
- 浏览器实地调研：微信/支付宝官方文档、36氪行业文章、woshipm产品文章、58同城实际操作流程
- 授权弹窗的交互设计、合规要求、用户预期管理

**不涉及内容**：
- 后端技术实现细节（加密算法、接口协议等）
- 服务商管理系统设计
- 具体开发排期和资源评估

# 分析

## 2.1 现状分析与问题诊断

### 核心问题：会员福利核销率低

| 模块名称 | 问题描述 | 问题类型 | 业务影响 | 严重程度 |
|---------|---------|---------|---------|---------|
| 福利领取 | 用户领取福利后无后续跟进，依赖用户自行联系服务商预约 | 流程断裂 | 核销率低，运营ROI差 | 高 |
| 信息传递 | 服务商无法主动联系用户，用户领完即忘 | 信息不对称 | 用户福利过期浪费，体验受损 | 高 |
| 用户触点 | 领取后无有效的再触达机制 | 触点缺失 | 错过最佳核销窗口期 | 中 |
| 合规风险 | 直接共享手机号给服务商缺乏用户授权环节 | 合规缺失 | 违反《个人信息保护法》，存在法律风险 | 高 |

### 根因分析

当前流程：用户领福利 → 获得券码 → 自行联系服务商 → 到店核销

核心问题在于"**领完福利到实际核销**"之间存在断层：
- 用户缺乏主动核销的动力（非紧急需求，容易被遗忘）
- 服务商缺乏触达用户的渠道（无法主动联系）
- 缺少一个"**授权桥接**"环节来连接用户和服务商

## 2.2 新需求与竞品分析

### （1）需求分析

| 用户故事 | 功能列表 | 优先级 |
|---------|---------|--------|
| 作为用户，我领取了洗牙福利，希望服务商能主动联系我安排预约，避免自己忘记 | 福利领取后弹出授权弹窗 | P0 |
| 作为用户，我想知道谁会联系我、什么时间联系、我的手机号会怎么用 | 弹窗展示服务商信息+用途说明+预计响应时间 | P0 |
| 作为用户，我希望授权是可撤回的，不想被骚扰时可以取消 | 授权管理页：查看/撤回授权 | P1 |
| 作为用户，我不希望每次领福利都弹授权，应该可以记住我的选择 | 授权有效期管理（单次/始终允许） | P1 |
| 作为服务商，我需要在规定时间内联系用户，否则影响我的评分 | 服务商SLA机制（响应时效+评价体系） | P1 |
| 作为运营，我需要知道授权转化率和核销率，评估功能效果 | 数据埋点：授权率、核销率、撤回率 | P2 |

# 结论与建议

## 核心结论

基于10个竞品平台的调研分析，美柚会员福利授权弹窗应采用 **"美团（商家授权机制） + 支付宝（授权管理页设计） + 58同城（透明服务商信息）"** 的融合方案。

## 方案建议

### 1. 授权时机
领福利后立即弹出授权弹窗（参考美团"咨询即授权"的即时性），但增加频率控制（同一服务商24h内不重复弹窗）。

### 2. 授权组件设计（两层授权）
- **第一层**：美柚自有说明弹窗 → 展示服务商名称、评价、预计响应时间（参考58同城经纪人卡片）、手机号用途
- **第二层**：用户同意后，若为微信小程序形态，调起微信原生 `getPhoneNumber` 获取加密手机号

### 3. SLA机制
授权成功后明确告知用户"服务商将在24小时内联系您"（参考平安预期管理），服务商超时未联系影响评分（参考58同城/美团SLA）。

### 4. 授权管理页
在App"隐私设置"或"会员中心"增加"我的授权"管理页：按服务商分组、标注授权时间+有效期、提供一键撤回按钮（参考支付宝授权管理页设计）。

### 5. 弹窗文案
从"是否授权"改为"帮您快速预约"的价值主张，降低用户决策阻力（参考美团"帮您了解服务详情"）。
